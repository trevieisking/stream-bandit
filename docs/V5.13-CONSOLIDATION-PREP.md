## Important bug lesson

The main bug pattern was not one single broken page. It was **old late patch scripts competing with newer late patch scripts**.

Symptoms seen:

- Version labels flickering between multiple versions.
- Browser tab switching versions.
- Sidebar badge duplicating or showing old versions.
- Extreme repeated text like `V5.11.2.2.2.2...`.
- Settings/Admin tab controllers fighting each other.
- Menu scroll-memory over-correcting and making the sidebar jump by itself.
- Aggressive layout script moving/labeling too many sections.

Prevention rule:

> Do not keep adding separate late `MutationObserver` / `setInterval` scripts that rewrite the same labels, tabs, menu, or layout. Proven fixes should be consolidated into one controlled path with hard duplicate guards.
