## Why this matters

The big ghost bug happened because multiple late scripts were all trying to own the same UI areas:

- browser title
- sidebar version badge
- Settings tabs
- Manager/Admin tabs
- manager layout labels
- sidebar/menu scroll

The fix was to stop loading competing scripts and keep one clear late controller path.

## Loader hygiene rule

Before adding a new script to `index.html`, check:

1. Does it rewrite version labels?
2. Does it create tabs?
3. Does it add repeated `MutationObserver` or `setInterval` logic?
4. Does it touch the same page area as an already-loaded file?
5. Can it be made idempotent with one unique wrapper ID and bound flags?

If the answer to 1–4 is yes, avoid it or merge it into the existing controller later.
