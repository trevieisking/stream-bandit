Quick checklist to copy

What Trevor tests

1
Page loads

Open the test page directly and make sure it does not black screen, loop, iframe the wrong page or crash.

2
Global look

Theme, colours, account/avatar and global helper status must match the rest of Stream Bandit.

3
Tabs and buttons

Click every visible tab and the main route buttons. Make sure they open the current pages.

4
Page role

Decide whether the page makes sense: editor, guide, tool, dashboard, concept or final-gate page.

5
Safety

For read-only pages, there must be no real save/upload/delete/publish/live controls.

What AI must scan

🔎
Route wrapper

Scan the old menu/wrapper route before creating or promoting anything.

🧭
Current target

Find the true current target page and any protected fallback.

🧩
Ownership

Do not duplicate ownership. Web Builder owns page/form layout, Profile Settings owns profile/avatar/banner, Live Readiness owns final gates.

🛡️
Safety

No secret keys, no schema changes, no live/index promotion and no destructive actions unless explicitly planned.

✅
Promotion

Only promote after Trevor passes the page. Then re-check the wrapper and save a checkpoint.
