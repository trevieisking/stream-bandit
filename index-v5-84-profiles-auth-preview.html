sb_profiles target behaviour:
1. Viewer can read own profile.
2. Creator can read own creator profile/channel data.
3. Admin can read review/profile records needed for moderation.
4. Site owner can access full platform builder/profile controls.
5. Profile updates must be limited to auth.uid() = id unless elevated.
6. Avatar file uploads should use stream-bandit-images or a future avatar bucket.
7. This V5.84 page performs no insert, update, delete, upload or auth request.
