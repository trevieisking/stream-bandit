Health Check role

What it is

A read-only dashboard that tells us if the app shell, helper scripts and important routes are reachable.

What it is not

It is not a repair page, migration page, policy editor, upload tool or live promotion page.

When to use

Use before Mux/Storage/Backup checks and before final release gates.

Header rule

Uses the same shell/header pattern as Live Readiness. No manual icon strip.

System health

This checks browser-visible helper state only. Supabase data quality belongs to Supabase Library, Review Queue or a dedicated quality page.

Route health

These checks only fetch known local pages. They do not write anything.
