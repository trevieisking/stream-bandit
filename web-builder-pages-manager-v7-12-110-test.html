Delete safety

This manager uses soft delete only. Hide / soft delete keeps the page row and content, sets status to hidden, unticks show in menu, and records hiddenAt. Restore brings it back to draft and show in menu.

Hard delete is intentionally not included in this test version.

Safety / rules

No schema change

Uses existing sb_site_pages only.

Navigation data location

Saves menu label, show/hide, sort order, home flag, description and hide/restore notes inside settings_json.navigation.

No shell edit

Includes protected shell helpers but does not edit shell files.

No hard delete

V7.12.110 does not delete rows or page blocks. It only hides/restores pages.
