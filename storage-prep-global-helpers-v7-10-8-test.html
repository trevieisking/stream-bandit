Overview

What this does

Turns an image file into a Supabase public URL for posters, logos, avatars, banners, page images and test artwork.

What this does not do

It does not decide which movie/channel/profile/page row gets the URL. Use the owning editor for that.

Why it helps

You do not need to know how to create the URL manually. Upload, copy, paste into the correct editor.

Safe scope

Authenticated upload only, no deletes, no policy changes, no service-role key.
