Creator group status

1. Submit Video

Creator creates a pending row in sb_submissions. Poster is uploaded first and video remains a Mux/video URL.

2. Review Queue

Admin checks the row and chooses the publish channel before approval.

3. Publish

Approve + Publish creates or updates a public row in sb_movies.

4. Library Edit

Supabase Library is the final place to clean titles, duplicate URLs, poster, channel, and metadata.

⬆️ Submission rules

Title

Use a clear title. Year in title is useful because year can be detected during publish.

Mux/video URL

Use a public playable Mux/HLS/video URL, not a private dashboard page.

Poster

Poster/artwork uses Supabase Storage. The URL must be visible before submitting.

Reason

Use the reason field to guide the reviewer. This remains in sb_submissions.

Status

New creator submissions start as pending.

Submission table

Submit Video writes to sb_submissions, not directly to sb_movies.

🧾 Review and publish rules

Pending

New submissions wait here until checked.

Queue
Approve + Publish

Creates or updates sb_movies, sets status='published', then marks the submission approved.

Go
Selected channel

Admin-selected channel writes to sb_movies.channel_id.

Channel
Approved only

Use only when you want to approve the submission without publishing a public movie row.

Status
Declined

Use declined when the submission should not move forward.

Stop
Library check

After publish, open Supabase Library and confirm poster, title, URL, channel, Details and Play.
