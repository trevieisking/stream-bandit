Dashboard tabs

Overview

User list

Show registered users with display name, email-safe display, plan, role, status and last activity later.

Plan view

Show the user plan and which Stream Bandit features are unlocked.

Permission view

Show can_watch, can_submit, can_create_channel, can_use_builder, can_review and can_publish style flags.

Safety view

Show warnings before changing roles, access, admin permissions or plan status.

Admin Workflow

  1. Open user profile.
  2. Check current plan and role.
  3. Review requested feature access.
  4. Preview permission changes before saving.
  5. Require backup/safety confirmation for admin/publish permissions.
  6. Record change in audit log later.
