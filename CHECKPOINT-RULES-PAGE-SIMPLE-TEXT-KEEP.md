# Stream Bandit — Rules Page Simple Text Keep

Checkpoint name:

`Rules Page - Simple Text Keep`

## Decision

Keep the Rules page simple.

## Reason

The Rules page is the simplest Stream Bandit page and should remain a plain, clear text/rules page.

It does not need:

- tabs,
- complex buttons,
- extra navigation,
- Supabase logic,
- page rebuilds,
- heavy visual polish.

## Current recommendation

Only update the Rules page later if the actual rules text needs improving or expanding.

Possible future rules additions:

- no illegal content,
- no adult/sexual content,
- no hate, harassment or threats,
- no graphic violence or dangerous content,
- no misleading kids content,
- only submit links/content you are allowed to add,
- admin can approve, edit or decline any submission.

## Protected areas

No code changes needed now.

Do not add functional changes just to tidy this page.

## Menu note

Rules can stay in Admin Tools for now, or later move into a Creator/Submit section if the sidebar gets a proper Creator group.
