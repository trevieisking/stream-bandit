Buddy tools index.

This page groups the Code Labs helper pages used during repair work. Read first, prepare one exact file, use Buddy Canvas for full-file replacements when patch recipes become fiddly, then use GitHub branch and pull request for file changes.

  2. Load full source in File Lab.
  3. Use Buddy Canvas for a full replacement.
  4. Branch and PR.
  5. Preview.
  6. Approve.
