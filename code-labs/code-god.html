Code God checks the saved action, repository, target path, branch, source provenance and complete file requirements that apply to that action. It cannot save, commit, merge, publish, deploy or delete.

Workflow

  2. For existing files, load the source from GitHub read-only.
  3. Run Code God.
  4. Resolve blocking findings.
  5. Continue to GitHub Writer and Codex.
