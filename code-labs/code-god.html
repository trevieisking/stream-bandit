Code God is the deterministic final gate. It checks the saved action, repository, target path, branch, source provenance and complete file requirements that apply to that action, including any separate CG Repair Lab candidate. It cannot save, commit, merge, publish, deploy or delete.

Workflow
  1. Save the intended GitHub action.
  2. Load verified source and, for Pro analysis, run CG Repair Lab.
  3. Run Code God.
  4. Resolve blocking findings.
  5. Continue to GitHub Writer and Codex only after PASS.
