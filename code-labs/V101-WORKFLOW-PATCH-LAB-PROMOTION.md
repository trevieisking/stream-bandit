# Code Labs V101 Workflow Promotion

Patch Lab is promoted from the Tools lane into the main Code Labs workflow.

New main flow:

1. Home
2. File Lab
3. Rescue Room
4. Workflow Hub
5. Patch Desk
6. Patch Lab
7. Preview + Test
8. Checkpoints
9. Repo Desk
10. GitHub Writer
11. GitHub Tracker

Patch Desk is still for full fixed files. Patch Lab is now the next step for exact line or small code patches before Preview + Test.

Safety: navigation only. No write connector changes. No Supabase schema/data changes. No Stream Bandit app page changes.
