Recommended workflow

1. Prepare

Save project details, choose the project and load the complete source.

2. Describe

Record the fault and preserve rules, then build the repair packet.

3. Repair

Use Buddy Canvas, Patch Desk or Patch Lab for the fixed output.

4. Test

Run Preview + Test and save a Checkpoint.

5. Review

Use Repo Desk and Code God before repository work.

6. Promote

Use GitHub Writer and Tracker for the branch and pull request.

Safety: This Home has no scripts, timers, observers or background requests.
