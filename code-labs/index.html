  1. Start with Setup.

Recommended workflow

1. Prepare

Save the project details, choose the project and load the complete source file.

2. Describe

Record what is broken, preserve working behaviour and build the repair packet.

3. Repair

Use Buddy Canvas, Workflow Hub, Patch Desk or Patch Lab for the fixed output.

4. Test

Run Preview + Test and save a Checkpoint before repository work.

5. Review

Use Repo Desk and Code God to verify the exact repository action.

6. Promote

Use GitHub Writer and Tracker for the branch, pull request and preview checks.

Safety: This Home contains no scripts, timers, observers, storage writes or background requests.

Safe next step
