2. Describe

3. Repair

Use Buddy Canvas, Patch Desk or Patch Lab to prepare the complete fixed output.

Run Preview + Test and save a Checkpoint before repository work.

5. Review

Use Repo Desk and Code God to check the proposed change safely.

6. Promote

Use GitHub Writer and Tracker for the branch and pull-request workflow.
